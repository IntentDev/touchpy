touchpy.InDats

.. py:class:: touchpy.InDats

   A container of InDat objects.

   
   

   .. rubric:: Attributes

   .. py:property:: count
      :type: int

      Returns the number of In DATs in the loaded tox.

   .. py:property:: names
      :type: list[str]
