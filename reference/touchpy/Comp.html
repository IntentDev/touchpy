touchpy.Comp🔗

class touchpy.Comp🔗
class Comp(
tox_path: str,
flags: CompFlags = CompFlags.INTERNAL_TIME_AUTO,
fps: int = 60,
)🔗

A TouchDesigner component loaded in a TouchEngine instance.

Attributes

property in_chops: InChops🔗

The In CHOPs of the currently loaded tox.

property in_dats: InDats🔗

The In DATs of the currently loaded tox.

property in_tops: InTops🔗

The In TOPs of the currently loaded tox.

property out_chops: OutChops🔗

The Out CHOPs of the currently loaded tox.

property out_dats: OutDats🔗

The Out DATs of the currently loaded tox.

property out_tops: OutTops🔗

The Out TOPs of the currently loaded tox.

property par: ParCollection🔗

The parameters of the currently loaded tox.

property rate: float🔗

Methods

Loads a .tox file, creates and initializes a TouchEngine Instance.

Parameters:

bool
